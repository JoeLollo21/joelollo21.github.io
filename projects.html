Academic/Research:

Baloney Detectives: A Virtual Misinformation Escape Room

Course: LIS 547, Design Methods, Winter 2023 (UW MLIS)
Presented At: MisInfo Day 2023 (UW iSchool, March 2023)
Keywords: Game Design, Value-Sensitive Design, Information Literacy, Game-Based Learning
Technologies Used: Twine (HTML/CSS game engine), itch.io, Figma

Abstract:
Inspired by Carl Sagan's 1984 essay, Baloney Detectives is a virtual escape room mimicking psychological and emotional responses to misinformation. The game aims to equip players with the necessary skills to identify misinformation online. The page on this site features an explanation of the design process, as well as most of the documentation (including the game host guide written as a final).

Re-Balancing Catan with Value-Sensitive Design

Course: LIS 547, Design Methods, Winter 2023 (UW MLIS)
Keywords: Systems Analysis, Value-Sensitive Design
Technologies Used: Canva, Medium

Abstract:
This blog post, which was a class assignment first, applies ideation and prototyping methods from Batya Friedman and David Hendry's toolkit for value-sensitive design to the board game Catan. The goal in the post is to explore ways to make the game more accessible to new players while maintaining the "difficulty spike" and challenge that makes experienced players return to the game.
